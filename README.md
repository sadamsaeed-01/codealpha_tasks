# Age Calculator

## Overview

Age Calculator is a simple web application that calculates a user's exact age in years, months, and days based on their date of birth.

## Features

* User-friendly interface
* Date of birth selection using date picker
* Calculates exact age in years, months, and days
* Prevents selection of future dates
* Responsive and attractive design

## Technologies Used

* HTML5
* CSS3
* JavaScript

## Project Structure

```text
Age-Calculator/
│
├── index.html
├── style.css
└── script.js
```

## How to Run

1. Download or clone the repository.
2. Open the project folder.
3. Run `index.html` in any web browser.
4. Select your date of birth.
5. Click the **Calculate** button to view your age.

## How It Works

* The user selects a birth date.
* JavaScript compares the selected date with the current date.
* The application calculates the difference in years, months, and days.
* The result is displayed on the screen instantly.
